# 按钮使用文档

| 属性           | 描述                    | 可选值                                      | 默认值      | 是否必填 |
| ------------ | --------------------- | ---------------------------------------- | -------- | ---- |
| type         | 按钮类型                  | `primary` `secondary` `success` `info` `warning` `danger` | 无        | 否    |
| size         | 按钮大小                  | `large` `medium` `small`                 | `medium` | 否    |
| nativeButton | 按钮类型                  | `button` `submit`                        | `button` | 否    |
| disabled     | native属性，按钮是否不可点击     |                                          |          |      |

| 事件     | 说明   |
| ------ | ---- |
| @click | 点击   |

> 其他说明：若要自定义大小，可使用style属性来覆盖原有大小信息
