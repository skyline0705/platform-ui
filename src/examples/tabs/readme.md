# 标签组件使用文档



## tabs文档

| 属性     | 描述                    | 可选值            | 默认值     | 是否必填 |
| ------ | --------------------- | -------------- | ------- | ---- |

| 事件          | 说明              |
| ----------- | --------------- |
| @tab-change | tab发生切换的时候触发的事件 |



## tab-panel文档

| 属性       | 描述        | 可选值            | 默认值     | 是否必填 |
| -------- | --------- | -------------- | ------- | ---- |
| active   | 当前tab是否选中 | `true` `false` | `false` | 否    |
| disabled | 当前tab是否禁用 | `true` `false` | `false` | 否    |
